Patches Content Warning's hosting system to always pretend to be a vanilla lobby, allowing cross-platform players to join modded servers

NOTE: This will only work properly for host-only mods, as console players cannot download mods themselves. Please use the mod responsibly and do not use it to trick players into joining modded games when they aren't expecting to.
